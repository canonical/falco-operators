# Falco operator

A [Juju][Juju] charm deploying and managing [Falco][Falco], an open-source runtime security tool
for physical and virtual machines. This subordinate charm simplifies the deployment, configuration,
and lifecycle management of Falco instances.

## Release

Falco operator uses the [falco binary][falco binary] and [falco plugins][falco plugins] directly
built from source. Please check the [RELEASE.md](./RELEASE.md) to learn more about the build and
release process.
